# Adiverse


> **Where Every Story Finds a Home.**

**Adiverse** is a premium, open, and distraction-free digital library and writing platform. Built as a serene alternative to cluttered, ad-heavy corporate platforms, it provides an elegant, 100% free sanctuary for readers and a highly focused web studio for writers. 

Adiverse puts the focus entirely back on raw human expression and pristine typography, categorizing content by "reading intent"—from quick 5-minute poems to immersive 45-minute novel sessions.

---

## Development Status: Active Build Phase

**Adiverse is currently under active development.** 
We are laying down a robust, serverless production architecture. The codebase is actively being refactored to support a highly scalable, frictionless public launch.

**Currently working on:**
*   Migrating the frontend to a serverless architecture using TanStack React Start.
*   Replacing legacy text editors with a modern, block-based rich-text engine (Tiptap) for pristine typography.
*   Implementing dynamic data loaders to prevent client-side routing crashes.
*   Wiring up a production-ready PostgreSQL database via Supabase for secure data storage and Row-Level Security (RLS).

## The Tech Stack (Under the Hood)
*   **Frontend:** React 18, TanStack Router, TanStack React Start
*   **Styling:** Tailwind CSS, Shadcn/ui, Radix UI
*   **Editor:** Tiptap (Rich Text)
*   **Backend / Database:** Supabase (PostgreSQL), Serverless API Functions
